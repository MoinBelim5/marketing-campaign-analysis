# Marketing Campaign Analysis

## Overview
This project analyzes marketing campaign performance using Python and Power BI.
The objective is to evaluate campaign effectiveness, customer behavior, and return on investment (ROI).

---

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- CSV Dataset

---

## Dataset
- Synthetic marketing campaign dataset created for analysis and learning purposes  
- Includes campaign details such as channel, age group, region, budget, impressions, clicks, conversions, and revenue  
- Raw data was cleaned and transformed using Python before visualization in Power BI

---

## Project Files
- marketing_campaign_data.csv (raw dataset)
- marketing_campaign_cleaned.csv (cleaned dataset)
- marketing_campaign_eda.ipynb (Python EDA)
- marketing_campaign_analysis.pbix (Power BI dashboard)

---

## Analysis Summary
- Performed data inspection and cleaning using Python
- Created key metrics such as CTR, Conversion Rate, ROI, and Campaign Duration
- Analyzed performance by channel, age group, and region
- Identified profitable and loss-making campaigns

---

## Power BI Dashboard
The dashboard presents:
- Total Revenue, Clicks, Conversion Rate, and ROI
- Channel-wise and region-wise performance
- Conversion trends by age group
